API Routes


/api/users
returns the users in insomnia

/api/thoughts
returns the thoughts in insomnia

/api/users/:id
returns a specifc, whole user in insomnia

/api/thoughts/:id
returns a specific thought

above are the required get Routes

POST, PUT, and DELETE routes for users and thoughts
